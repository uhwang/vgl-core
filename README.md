# vgl-core
VGL package for distribution  
> [!NOTE]
> Python version: 3.11.4

1. How to create libvgl package
   - pip install wheel
   - python setup.py bdist_wheel

2. Install libvgl package
   - pip install dist\libvgl-0.1-py3-none-any.whl

3. Usage
   - Run python
   ```Python
   import libvgl as vgl
   vgl.pixel_grid()
